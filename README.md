# Node.js Authentication Mini Project

A simple user authentication system built with Node.js, Express, MongoDB, JWT, and bcrypt.

---

## Features
- User registration with hashed passwords  
- User login with JWT authentication  
- Cookies for session handling  
- Logout functionality  
- TailwindCSS UI  
- Mongoose data modeling  

---

## Tech Stack
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- EJS (for frontend views)  
- bcrypt (password hashing)  
- JWT (authentication)
